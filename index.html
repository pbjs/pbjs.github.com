As a JavaScript developer you know how exhausting it is to deal with DOM manipulation,
different browser versions, multiple devices and random code syntax.

There was no standard to deal with these issues before - until a few years ago when frameworks
like jQuery or Prototype appeared. We gladly payed the price by loading these large libraries,
as they helped us to easily create dynamic web pages.

Nowadays we depend on these - even though they becoming bloated or got outdated.

Since more people get interested in creating large web applications,
we needed a solution which offers a polished, fast & maintable structure.

The Users experience should be considered as well - regarding smooth animations and fast rendering on all devices.

This framework inherits most of the common features, while it kept the small size
and supports best practice in front-end development.
